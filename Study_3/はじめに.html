【作成するアプリ】
今回作成するアプリは、足し算、引き算、掛け算などの計算を実際にすることができる電卓アプリです！
各数字ボタン：電卓内部の処理に向けて、数字の値を渡します
演算子ボタン（+, -, *, /）：数値同士の演算内容を定めることができます
イコールボタン（=）：入力した数値を演算子に従って計算することができます

書いてあるコードの内容については、今は理解できなくても問題ありません！

【手順】

０、今回もStudy_1の手順と同様に模写をしていきますが、

Visual Studio Codeのインストール手順

■以下のサイトの「今すぐダウンロード」ボタンをクリック
https://azure.microsoft.com/ja-jp/products/visual-studio-code/

■PCにダウンロードされたインストーラーを開く
　※インストーラーのファイル名は、VSCodeUserSetup-x64-1.55.1.exeのような形

■インストールウィンドウが開くので、基本的には「はい」や「Yes」で進めていく
※こちらのサイトが参考になります

■以下のイメージがVisual Studio Codeの画面になりますが、色が付いていて見やすいですし、
　様々な機能が付いています♪

１、まず、パソコンの中から「メモ帳」（Macの場合は、「テキストエディット」みたいです）を開きます

　 【重要】０手順にてVisual Studio Codeをインストールしている場合は、「メモ帳」を「Visual Studio Code」に置き換えてください
　　Visual Studio Code操作方法参考

メモ帳を開く手順

例１：

■画面左下のWindowsロゴをクリック


■メニュー上部の「すべてのアプリ」をクリック


■表示されたアプリ内から、「Windowsアクセサリ」をクリック


■メモ帳をクリック
例２：

■キーボードの「Windowsロゴキー + R」を同時に押す


■表示された入力欄に「notepad」と入力し、Enterキーを押す

こちらのサイトも参考になります（https://office-hack.com/windows/windows10-notepad/）

２、次にテキストファイルとして保存します

テキストファイルとして保存する手順

■開いたメモ帳を選択した状態で、左上にある「ファイル」をクリック


■表示されたメニューから「名前を付けて保存」をクリック


■名前を付けて保存するためのエクスプローラーが開きますので、左上の「新しいフォルダー」をクリック


■作成された新しいフォルダーに「プログラミング勉強用フォルダ」など好きな名前を付け、
    そのフォルダをダブルクリックし、フォルダ内に入る

■ファイル名欄に「study_3.html」を入力し、「保存」ボタンをクリック


参考サイト：https://office-hack.com/windows/windows10-notepad/

３、送付した「study_3.html」の記載内容を、保存して頂いた「study_3.html」に模写します

【重要】自分で記載内容を打ち込むことが大切なので、コピー＆ペーストは今回は無しでお願いしますm(_ _)m


模写する手順

■タスクバーに表示されているメモ帳を右クリックし、表示されたメニューの中から「メモ帳」をクリック


■新しいメモ帳が開くので、見本として受け取った「study_3.html」のファイルをクリックしたままでメモ帳のウィンドウへ移動させ、クリックを離す


■開いた見本を見ながら保存して頂いた「study_3.html」へ書き写す



■書き写し終わったら、「Ctrl + S」、もしくはメモ帳左上の「ファイル」 -> 「上書き保存」をクリックして保存します
